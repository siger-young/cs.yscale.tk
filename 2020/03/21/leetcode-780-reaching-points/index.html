LeetCode 780 Reaching Points

Description

A move consists of taking a point (x,y)(x, y)(x,y) and transforming it to either (x,x+y)(x, x+y)(x,x+y) or (x+y,y)(x+y, y)(x+y,y).
Given a starting point (sx,sy)(sx, sy)(sx,sy) and a target point (tx,ty)(tx, ty)(tx,ty), return True if and only if a sequence of moves exists to transform the point (sx,sy)(sx, sy)(sx,sy) to (tx,ty)(tx, ty)(tx,ty). Otherwise, return False.

Solution

纯辗转相减 (TLE)

由于变换是将 (x,y)(x,y)(x,y) 变为 (x,x+y)(x,x+y)(x,x+y) 或者 (x+y,y)(x+y,y)(x+y,y), 可以知道对 (x′,y′)(x',y')(x′,y′) 而言，两个数 x′x'x′, y′y'y′ 中较大的充当 x+yx+yx+y, 较小的充当 xxx 或 yyy, 根据位置可以唯一确定, 记 S(x,y)S_{(x,y)}S(x,y)​ 表示 (x,y)(x,y)(x,y) 的上一级, 则有

S(x,y)={(x−y,y),x>y(x,y−x),y>x.S_{(x,y)}=\begin{cases}(x-y,y)&,x>y\\(x,y-x)&,y>x.\end{cases} S(x,y)​={(x−y,y)(x,y−x)​,x>y,y>x.​

可以想到从终点 (tx,ty)(tx,ty)(tx,ty) 不断逆推上级, 记当前推到的点为 (xi,yi)(x_i,y_i)(xi​,yi​), 如果 (xi,yi)=(sx,sy)(x_i,y_i)=(sx,sy)(xi​,yi​)=(sx,sy), 就说明可行, 否则如果推到边界情况时（容易想到的一个边界是 xi=0∨yi=0x_i=0\lor y_i=0xi​=0∨yi​=0, 实际上有更紧的边界, 即 xi<sx∨yi<syx_i<sx\lor y_i<syxi​<sx∨yi​<sy）, 就说明不可行, 然而这一方法会 TLE.

辗转相除

辗转相除即

S(x,y)={(x mod y,y),x>y(x,y mod x),y>x.S_{(x,y)}=\begin{cases}(x \bmod y,y)&,x>y\\(x,y \bmod x)&,y>x.\end{cases} S(x,y)​={(xmody,y)(x,ymodx)​,x>y,y>x.​

实际上我一开始想到了纯辗转相除, 但是下面这个样例让我选择了纯辗转相减

如果使用纯辗转相除找上级, 那么 (25,7)(25,7)(25,7) 的上级就是 (4,7)(4,7)(4,7) 这一步就已经出现了问题, 一个自然的想法是对 (sx,sy)(sx, sy)(sx,sy) 和 (tx,ty)(tx, ty)(tx,ty) 分别辗转相除至一个“根”, 但这样做也是有问题的, 如下例.

可以发现对它们进行一次辗转相除都得到上级 (4,7)(4,7)(4,7), 而实际上这是行不通的.

为了发现思路, 我们不妨对上述两个样例画成图.

在这里绘制时采用的原则是, (x,y)(x,y)(x,y)变换为 (x+y,y)(x+y,y)(x+y,y) 时走左结点, (x,y)(x,y)(x,y)变换为 (x,x+y)(x,x+y)(x,x+y) 时走右结点, 利用辗转相除法寻找上级, 观察上图可以发现几个事实:
引理 1 从 (x,y)(x,y)(x,y) 变换到 (x+y,y)(x+y,y)(x+y,y) 再变换到 (x+y,x+2y)(x+y,x+2y)(x+y,x+2y), 对 (x+y,x+2y)(x+y,x+2y)(x+y,x+2y) 利用辗转相除法寻找上级, 可以找到 (x+y,y)(x+y,y)(x+y,y). 更普遍的, 辗转相除法会寻找离自己最近的改变变换模式的点（这里 (x+y,y)(x+y,y)(x+y,y) 之前采用的是“左”模式, (x+y,y)(x+y,y)(x+y,y) 之后采用的时“右”模式）.

推论 从 (x,y)(x<y)(x,y) (x<y)(x,y)(x<y) 变换到 (x+y,y)(x+y,y)(x+y,y) 一直变换到 (x+ky,y)(x+ky,y)(x+ky,y), 对最后一个点辗转相除寻找上级, 可以找到 (x,y)(x,y)(x,y).

引理 2 若 (sx,sy)(sx,sy)(sx,sy) 能变换到 (tx,ty)(tx,ty)(tx,ty) 那么两个点在一条链上.

结论 (sx,sy)(sx,sy)(sx,sy) 能变换到 (tx,ty)(tx,ty)(tx,ty), 当且仅当对于 (tx,ty)(tx,ty)(tx,ty) 的所有上级 (xi,yi)(x_i,y_i)(xi​,yi​) 而言, 必存在一个 (xi,yi)(x_i,y_i)(xi​,yi​) 使得 xi=sx∧xi∣(sy−yi)x_i=sx\land x_i\mid(sy-y_i)xi​=sx∧xi​∣(sy−yi​) 或 yi=sy∧yi∣(sx−xi)y_i=sy\land y_i\mid(sx-x_i)yi​=sy∧yi​∣(sx−xi​).
